## Steps

1. 从主界面点击 **"协会"** 进入公会页面
2. # ── 应援 ──
3. 找到并点击 **"应援"** 入口
4. # 查看今日剩余应援次数（如0/3），如有剩余次数执行应援
5. # ⚠️ 应援页面有 **上/下两个"每日应援"按钮**，外观几乎相同！
6. # → `adb_tap('每日应援', nth=1)` = **上方按钮 = 普通应援**（免费），✅
7. # → `adb_tap('每日应援', nth=2)` = **下方按钮 = 全力应援**（消耗50钻石），🚫 绝对不要点！
8. # → 如果 nth 匹配不确定 → magnify() 放大了再确认
9. # 应援完成后确认"今日应援次数已用完"
10. adb_back() 返回协会主界面
11. # ── 灵感碰撞 ──
12. 点击 **"灵感碰撞"**
13. # → 选择进度没满的提交；都满了就随便提交
14. # → 重点是 **"今日已提交"** 一定要达到5次
15. # → ⚠️ **"开启企划"** 按钮亮了必须点击，有💎钻石奖励！
16. adb_back() 返回主界面
17. subtask_done('lifemakeover-guild', '协会应援+灵感碰撞已完成（普通应援未消耗钻石）')

## Pitfalls

- 🚫 **两个"每日应援"按钮**：`adb_tap('每日应援', nth=1)` 点上方普通应援，永远不要用 nth=2
- 🚫 OCR可能把"应援"识别为"应缓"，用"应"字模糊搜索也能定位
- 协会应援次数每日有限（通常2-3次），优先完成以免浪费
- "开启企划"亮了点了有钻石奖励，不要漏掉
