## Steps

1. 从主界面点击右侧 **"日程"** 进入日程页面
2. # 日程页面：顶部显示活跃度进度条"完成每日日程可获得钻石等奖励"，下方为任务清单
3. # ⚠️ 页面上有多个同名"前往"按钮 → 用 nth 参数精确选择：
4. # → `adb_tap('前往', nth=1)` = 签到，`nth=2` = 思绪漫步，`nth=3` = 普通主线，以此类推
5. # → 如果 nth 失效 → magnify() → tap_magnified() 回退方案
6. # ── 任务1：每日签到 ──
7. # 点击第1个"前往"（y~560） → 进入签到页面（月度日历格子）
8. # 签到页面是一个月度日历：已签到的格子显示打勾✓，**今天的格子有红点/未领取标记**
9. # → 点有红点的格子 → 弹出奖励弹窗 → adb_back() 关闭 → 签到完成
10. # → 不要被前面打了勾的格子骗了——任务进度0/1就说明今天的还没签
11. adb_back() 返回日程
12. # ── 任务2：完成1次思绪漫步 ──
13. # 点击第2个"前往"（y~710） → 进入思绪漫步页面 → 点击"每日练习"
14. # ⚠️ 思绪漫步动画耗时约45秒 → 等待完成后领取奖励 → adb_back() 返回日程
15. # → 可以先做其他任务再回来领奖励，但当天一定要领
16. # ── 任务3：在时尚对决中完成挑战 ──
17. # 滚动日程页面找到"在时尚对决中完成X次挑战" → 点击该任务的"前往"跳转
18. # ⚠️ 如果看不到时尚对决任务，需要向下滑动日程页面才能找到
19. # ── 阶段A：搭配参赛服（如果还没有参赛服或没有搭配过）──
20. # 时尚对决界面显示"搭配参赛服"区域，下方有属性栏目（性感/清纯/华丽等）
21. # → 点击任意一个属性栏目 → 进入搭配衣服界面
22. # → 点击"推荐搭配" → 系统自动搭配 → 点"下一步" → 参赛服准备完成
23. # → 返回时尚对决主界面
24. # ── 阶段B：挑战次数计算 ──
25. # 每日挑战次数（根据星期几）：周一15次 / 周日5次 / 其余每天3次
26. # → 用 `[系统上下文]` 中的星期几判断当天需要打几次
27. # ── 阶段C：选择对手 ──
28. # 第一次对决前当前分数为0 → 随便选一个对手开始
29. # 打完一次后有分数了 → 选**前面两个对手**（段位积分比自己高的）
30. # → 打高分段对手可以获得更多积分
31. # → ⚠️ **打高分段对手，不是打低分段**
32. # → 如果没有合适的对手 → 使用每日免费刷新（3次/天）刷新对手列表
33. # ── 阶段D：执行挑战 ──
34. # 选择对手后 → 进入对决 → 系统自动使用参赛服对战 → 等待结果
35. # → 重复直到完成当天要求的次数
36. adb_back() 返回主界面
37. # ⚠️ 签到/思绪漫步/时尚对决做完后不要急着点"一键领取"——等 lifemakeover-farm 完成所有刷关卡后再统一领取活跃度
38. subtask_done('lifemakeover-schedule', '已完成签到+思绪漫步+时尚对决（活跃度待farm完成后统一领取）')
39. adb_back() 返回主界面

## Pitfalls

- 🚫 **绝对不要用 `adb_tap('前往')` 不加 nth——5个同名按钮必然点错！** 用 `adb_tap('前往', nth=N)` 选第N个
- 🚫 **签到页面别被打了勾的格子骗了**：前面几天打了勾=历史已签，今天的格子才有红点。"已连续登录N天"不等于今天签到了——日程显示 0/1 就是没签。哪怕日历上全是勾，只要日程是 0/1，就还有一个带红点的格子等着你点。
- 🚫 日程页面标题含"钻石"两字是正常的奖励描述，不要犹豫操作
- 🚫 思绪漫步耗时45秒，可以等也可以先做其他任务
- 🚫 **时尚对决打高分段对手（前两个），不是低分段**——打高分涨分快
- 🚫 时尚对决每日免费刷新3次，精打细算用
- 🚫 **签到+思绪漫步+时尚对决做完后不要一键领取活跃度**——留给 lifemakeover-farm 在所有关卡刷完后统一领取
