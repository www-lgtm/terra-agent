## Steps

1. # 入口：主界面 → 闪亮之旅 → 代言女王；或日程页面点击"前往"跳转
2. # ── 每日任务：每日宣传 ──
3. # 点击"每日宣传" → 再点"开始宣传" → 领取奖励弹窗 → adb_back() 关闭 → 返回
4. # ── 进阶任务：代言挑战（四个街区）──
5. # 代言分为四个街区（不同属性要求），对每个街区重复以下：
6. # 1）进入街区 → 拖动查看地图上👑空位的店铺 → 点击该店铺
7. # 2）查看代言详情：分数要求 + 预估搭配分数
8. # 3）点击"快速代言"开始挑战 → 等待评分 → 查看结果
9. # 4）领取奖励 → 继续下一个店铺
10. # → 从低等级店铺开始打（低等级失利=后面高等级也打不过，停了就行）
11. # → 四个街区都尽量尝试一遍
12. # ── 小镇探索与拾取（独立步骤，不受代言结果影响）──
13. # ⚠️ 即使代言挑战失利，一键拾取仍然要执行
14. # → 在小镇地图上点击"一键拾取"按钮收集所有建筑产出
15. subtask_done('lifemakeover-endorse', '每日宣传+代言挑战+一键拾取已完成')

## Pitfalls

- 👑数量 = 该店铺的关卡阶段数：低级阶段1关、中级2关
- 分数要求与关卡等级成正比，从低到高打，低等级失利就停
- **一键拾取和代言是独立的——代言打不过不影响一键拾取，两者都要做**
