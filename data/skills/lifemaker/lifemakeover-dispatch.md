## Steps

1. 从主界面点击 **"闪亮之旅"** → 点击 **"日常事件簿"** 入口
2. # 先查看是否有已到期可领取的事件（显示"领取"按钮）
3. # → 事件三种状态："领取"（可领） / "已派遣"（计时中） / "派遣"（可派遣）
4. # → 优先点击 **"一键领取"** 领取所有已完成事件的奖励
5. # 选择显示"派遣"的新事件 → 点击对应的 **"派遣"** 按钮
6. # → 查看事件详情：能力要求、奖励掉落、耗时
7. # → 选择属性匹配的SSR羁绊提高评价
8. # 点击底部 **"一键派遣"** → 再点击 **"确认派遣"**
9. # 重复直到四个事件全部派遣完
10. subtask_done('lifemakeover-dispatch', '事件派遣已完成（已领取+已派遣4个事件）')

## Pitfalls

- 如果已派遣事件还没到期，跳过该事件
- 优先SSR羁绊提高评价，属性匹配比稀有度更重要
