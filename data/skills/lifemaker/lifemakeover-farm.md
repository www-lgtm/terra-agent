## Steps

1. # ── 普通主线 10 次 ──
2. # 主界面 → 闪亮之旅 → 主线之旅 → 普通难度
3. # 选择一个已解锁快速搭配的关卡 → 点击"搭配十次"一次性完成10次
4. # ⚠️ 扫荡结果弹窗关闭方式：优先 adb_back()，不要点 X 按钮（位置不固定经常点不中）
5. adb_back() 返回关卡列表 → adb_back() 返回闪亮之旅
6. # ── 困难主线 10 次 ──
7. # 在主线之旅页面点击底部"困难"切换难度
8. # ⚠️ 困难主线必须刷以下固定关卡（第二章免活复刻材料）：
9. # → 2-3, 2-4, 2-5, 2-7, 2-8, 2-9, 2-10, 2-12
10. # → 跳过 2-6 和 2-11（不掉落目标材料）
11. # 导航到第二章：点击顶部章节标题（带▼箭头）→ 在章节选择器中点"第二章"
12. # ⚠️ 关卡列表需要左右滑动才能看到全部关卡
13. # → 滑动方向说明：向左滑动 = 看后面的关卡（2-7→2-8→2-9），向右滑动 = 看前面的关卡（2-5→2-4→2-3）
14. # → 滑不动说明到头了，换方向滑或直接选可见的关卡开始
15. # ⚠️ OCR问题："关卡2-4"可能被模糊匹配为"关卡2"然后点到2-3
16. # → 优先用 magnify → mark_position → tap_magnified 精确定位
17. # 每个关卡每天有 3 次挑战限制，困难关卡用"搭配三次"一次性刷完
18. # 10 次分配：2-3×3 + 2-4×3 + 2-5×1 + 2-9×3 = 10（或按实际情况调整，确保总分到10）
19. # 每次刷完后：adb_back() 关闭结果弹窗 → 回到关卡列表 → 继续下一个
20. # 困难主线10次完成后 → adb_back() 返回闪亮之旅
21. # ── 心意之期 10 次 ──
22. # 闪亮之旅页面点击"心意之期"入口
23. # ⚠️ 页面上"今日心意之期次数 X/10 次" → X 是**剩余次数**，不是已用次数！
24. # → X=10 表示今天一次都还没刷，还有10次可以刷
25. # → X=0 才表示今天刷完了
26. # 心意之期页面展示多个套装，每个套装对应一个关卡
27. # ⚠️ 不要随便选关卡——必须优先刷**还没成套的套装**：
28. # 1）浏览心意之期页面上的套装列表
29. # 2）查看每个套装的收集进度（如"3/6"=已收集3件缺3件，"6/6"=已成套）
30. # 3）优先选择**进度未满、缺件最多**的套装，点击进入该关卡
31. # 4）已解锁快速搭配 → 点击"搭配多次" → 缺几件刷几次
32. # 5）未解锁 → 先手动过一次拿到"精彩"评价 → 再搭配多次
33. # 6）重复直到累计完成 10 次心意之期
34. # → 如果所有套装都齐了 → 选任意已解锁快速搭配的关卡完成 10 次
35. # 每次刷完后：adb_back() 关闭结果弹窗
36. # ── 回到日程页面领取活跃度 ──
37. # 三个任务全部完成（普通10+困难10+心意10）→ adb_back() 返回主界面 → 点击"日程"
38. # → 进日程页面 → 确认活跃度已达标 → 点击"一键领取"领取活跃度奖励
39. # → 关闭领取弹窗（adb_back()）→ adb_back() 返回主界面
40. # ── 国服（未来）：设计室驱动刷法 ──
41. # 先去主界面 → 设计室 → 查看正在制作的图纸 → 查看材料缺口
42. # 缺的材料旁边有"前往"按钮 → 点击跳转到对应关卡 → 搭配多次直到材料充足
43. subtask_done('lifemakeover-farm', '已完成普通主线10次+困难固定关卡10次+心意之期10次，活跃度已领取')

## Pitfalls

- 🚫 **困难主线必须刷指定关卡列表**（2-3~2-12，跳过2-6和2-11），不要随便选
- 🚫 **心意之期必须优先刷未成套的套装关卡**——进页面后先看各套装收集进度
- 🚫 **关闭结果弹窗用 adb_back()**，不要点屏幕上的 X（经常点不中位置）
- 🚫 **OCR 模糊匹配**：adb_tap('关卡2-4') 可能匹配到"关卡2"点到 2-3 → `adb_tap('关卡2-4')` 如果点错，用 magnify + tap_magnified
- 🚫 困难关卡每天每关只有 3 次限制，注意分配
- 🚫 **活跃度要等三个任务全部刷完再领**，不要提前一键领取
- 🚫 如果关卡显示 NEW → 先手动过一次拿"精彩"评价 → 再搭配多次
- 🚫 **进了剧情/故事/对话画面 = 你点错了「开始故事」，不是扫荡！** 立刻 adb_back() 退出，回到关卡列表重新选关卡，然后**只看「搭配一次/三次/十次」**这些按钮，永远不要点「开始故事」
- 🚫 **只选标记了「完美」的关卡**，没有「完美」标记的不碰（会触发剧情），除非所有关卡都没标记
