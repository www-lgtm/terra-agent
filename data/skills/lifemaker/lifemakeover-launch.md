## Steps

1. 在手机桌面找到 **biubiu加速器**（蓝绿色图标）并点击打开
2. # biubiu可能先弹开屏广告（左下角"跳过3"倒计时），等待倒计时结束或点"跳过"
3. # 进入biubiu首页后，找到"以闪亮之名（新加坡服）"，点击旁边的 **"加速"** 按钮
4. # ⚠️ 注意：首页有2处"加速"文字 → 用 `adb_tap('加速', nth=2)` 点游戏旁边的加速按钮（nth=1是顶部标签栏）
5. # → 如果 nth 失效 → magnify() → tap_magnified() 回退
6. # ── 陷阱A：时长不足需要看广告 ──
7. # 如果弹出"今日福利已就绪 — 立即获取" → 说明加速时长不足
8. # → 点击"立即获取"进入"看广告领时长"页面
9. # → 点击"获取时长"播放广告（约30秒），等待倒计时 → 左上角X关闭
10. # → 重复看3个广告直到获得24小时
11. # → 广告中间弹出"浏览5秒 提前领奖"诱导弹窗 → 点击 **"残忍离开"** 或X关闭 → 继续等待倒计时
12. # 加速成功后biubiu自动拉起游戏
13. # ── 游戏启动和加载 ──
14. # 游戏启动过程：黑屏 → 开屏Logo → 标题画面（显示"Life Makeover"和底部"点击开始"）
15. # 黑屏/加载进度条/初始化资源提示"此过程不消耗流量" = 正常过渡，**不要操作，耐心等待**
16. adb_tap({'target': '点击开始'}) # 标题画面需要手动点击才能进入主界面
17. # ── 关闭登录弹窗 ──
18. # 游戏进入主界面后会依次弹出多个活动公告/商城推广弹窗
19. # 常见弹窗：活动公告（"蔓花约"/"春原洋语"）、奇趣盲盒、商城礼包（"森息春谕"/"甜心有咩"/"春风寄梦"）、变装物语
20. # 关闭方式：**adb_back()** 逐个关闭，不要手动点X（部分弹窗X在底部中央不在右上角，容易点错）
21. # 重复 adb_back() 直到没有弹窗遮挡，确认到达游戏主界面
22. # 主界面标志：角色立绘 + 底部菜单栏（设计室/家园/衣橱/羁绊/追光） + 右侧菜单（日程/协会等）, # 趁着还在主界面上截图推送
23. notify_with_screen("已进入以闪亮之名，开始执行日常任务")
24. subtask_done('lifemakeover-launch', '游戏已启动并到达主界面，弹窗已全部关闭')

## Pitfalls

- 🚫 **"还未安装该游戏" = 广告误报，不是真的没安装！** 点击"暂时不用"关闭弹窗后继续点加速，绝对不要判定为"游戏未安装"然后去桌面搜索或ask_user
- 🚫 **广告中间弹窗"浏览5秒 提前领奖"不要点"提前领奖"**（会跳转网页广告），点在左上角❌关闭
- 🚫 **biubiu首页"加速"有两处同名文字**：用 `adb_tap('加速', nth=2)` 选游戏旁边的，nth=1是标签栏
- 🚫 游戏加载黑屏/进度条/初始化 = 正常过渡，不要操作，不要问用户
- 🚫 标题画面必须手动点击"点击开始"才能进入主界面，游戏不会自动进去
- 🚫 关闭弹窗优先用 adb_back()，不要尝试定位弹窗上的X按钮（位置不固定，经常点错被系统拦截）
- 🚫 **biubiu 广告地狱逃生**：如果看广告 → 坚持退出 → 获取时长 → 又弹广告，循环超过 5 轮仍没进游戏 → 立即放弃 biubiu，用 adb_launch_app('com.bluepoch.bluepoch.reversenineninetynine') 直启游戏。biubiu 的广告系统就是设计来困住用户的，不要跟它耗。
