# Life Makeover (以闪亮之名) Icon Templates

Place PNG template images here for icon/button recognition via template matching.

### 采集

```bash
python scripts/capture_templates.py
```

框选图标 → 输入文件名（中英文都行）→ 输入含义 → 保存。

### 建议的图标

| 文件名 | 含义 |
|--------|------|
| 返回箭头 | 左上角的返回箭头图标 |
| 日程 | 主界面右侧日程按钮（日历图标） |
| 协会 | 主界面右侧协会按钮 |
| 关闭按钮 | 弹窗右上角×关闭按钮 |
| 确认勾选 | 弹窗底部对勾✓按钮 |
| 一键领取 | 日程页面底部"一键领取"按钮 |
| 前往 | 日程页面任务旁的"前往"按钮 |
| 每日宣传 | 代言女王页面"每日宣传"按钮 |
| 开始宣传 | 每日宣传弹窗中的"开始宣传"按钮 |

你可以随意增减，采集时输入文件名和含义，LLM 就能自动识别。
