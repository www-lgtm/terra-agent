# Arknights Icon Templates

### 采集

```bash
python scripts/capture_templates.py
```

框选图标 → 输入文件名（中英文都行）→ 输入含义 → 保存。

### 建议的图标

| 文件名 | 含义 |
|--------|------|
| 精英2徽章 | 干员已精英化2阶段的金色翼形徽章 |
| 精英1徽章 | 干员已精英化1阶段的银色箭头徽章 |
| 精英0 | 等级数字旁无徽章（框选同一位置空白） |
| 任务完成 | 任务面板亮色棋子 |
| 任务未完成 | 任务面板暗色棋子 |
| 代理指挥已勾选 | 代理指挥方框内有X/数字 |
| 代理指挥未勾选 | 代理指挥空白方框 |
| 返回箭头 | 左上角的返回箭头图标 |

你可以随意增减，采集时输入文件名和含义，LLM 就能自动识别。
